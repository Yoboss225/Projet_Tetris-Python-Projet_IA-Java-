JEU TURNER TEEN ( Tétris )

Principe du jeu

Tétris se joue sur un quadrillage, appelé la matrice, possédant une gravité (du haut vers le bas). Des formes géométriques tombent depuis le haut de l’écran. Le joueur doit contrôler la chute de ces pièces. Il n’est pas possible de ralentir la chute des pièces. Le joueur doit ainsi empiler les pièces sur le bas de l’écran. Les pièces tombent une par une, et fusionnent avec le reste des pièces tombées une fois parvenues en bas. 
Le but du jeu est de former des lignes : lorsqu’un bloc termine une ou plusieurs lignes (occupant alors toute la largeur de la matrice), les lignes concernées disparaissent. Les pièces se trouvant au-dessus de la ligne disparue sont alors décalées vers le bas. En complétant des lignes, on peut donc empêcher la pile de pièces d’atteindre le haut de l’écran : si cela se produit, la partie est alors terminée. 

Les pièces

    • La pièce I, appelée aussi barre ;
    • La pièce 0, appelée aussi carré ;
    • La pièce T ;
    • Les pièces J et L, symétriques l’une de l’autre ;
    • Les pièces S et Z, symétriques l’une de l’autre.

Les commandes

1.les touches de direction:

      >>> Pour déplacer la pièce vers la gauche 

      >>> Pour déplacer la pièce vers la droite

      >>> Pour accélérer la chute de la pièce

2.La touche CTRL
               
      >>> Pour effectuer la rotation de la pièce suivant ses différentes positions
